# СПБАУ. Прикладная математика и физика. 2025-2026. Конспекты лекций

В данном репозитории хранятся конспекты, написанные по мотивам лекций бакалавриата ПМФ СПБАУ.

Для исправления опечаток создайте pull request с отредактированным typst файлом.
При желании можно также прикрепить pdf файл, но процесс так устроен, что всякий раз при изменении какого-либо typst файла происходит автоматическая перекомпиляция соответствующей pdf-ки.

Некоторые конспекты заменены на аналогичные с других вузов (мне лень их техать, но pull request не в бане)
